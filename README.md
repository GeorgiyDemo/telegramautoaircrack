## Configure setup
Replace this strings in autopwn.py:
* TELEGRAM_TOKEN
* TELEGRAM_CHAT_ID

If you want to use SOCKS5-proxy:
* proxy_url
* username
* password

## Requirements

* [python-telegram-bot](https://github.com/python-telegram-bot/python-telegram-bot)
